Principais formtatações

Negrito/destaque

Para negrito devemos utilizar a tag <strong>

Nesta frase isso ficara em negrito!

Itálico/Ênfase

Para usar itálico, utiliza-se a tag <em>

Agora isso ficará em itálico!

Texto marcado/grifado

Para grifar texto, usa-se a tag <mark>!

Esse texto vai ter uma parte grifada!

Texto grande e pequeno

Para grande usa-se <big>, já para pequeno usa a tag <small>

Esse texto tem parte grande

Já essa parte vai ficar pequena!

Texto Inserido e deletado

Para texto inserido, usamos <ins> e para deletado usamos <del>

Essa parte ficara excluída!

Já essa parte ficara inserida!

Sobrescrito e Subscrito

Para partes sobrescritas usamos <sup> e para subscrito usa-se <sub>
